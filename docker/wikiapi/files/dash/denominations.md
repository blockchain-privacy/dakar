# Dash Denominations

Denominations are predefined amounts of currency. To increase privacy,
funds which are to be mixed will get split into a set of denominations
via an [origin transaction](dash/originTransaction.md).

Dash defines the following denominations: `10.0001, 1.00001, 0.100001, 0.0100001, 0.00100001`.

See more [here](https://dashcore.readme.io/docs/core-guide-dash-features-privatesend#wallet-preparation).
