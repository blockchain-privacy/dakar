# Dash Collateral Creation Transaction

A collateral creation transaction is created during the [mixing process](dash/mixingProcess.md), to prepare funds,
which are later spent via a [collateral payment transaction](dash/collateralPaymentTransaction.md).

Collateral creation transactions spend funds from an [origin](dash/originTransaction.md), [mixing](dash/mixingTransaction.md) or another collateral creation transaction.
