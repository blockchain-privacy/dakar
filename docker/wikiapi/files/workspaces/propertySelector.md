# Property Selector

The Transaction Property selector is a powerful search feature that enables users to find transactions based on specific criteria. Users can search for transactions using the following properties:

- **Time**: Filter transactions based on the date and time they occurred.
- **Transaction Type**: Search for transactions by their type, such as standard transactions, CoinJoin transactions, or other categories.
- **Amount**: Find transactions that have a specific input or output amount range.
- **Amount Sum**: Find transactions that have a specific input or output amount sum range.

The selector stores all resulting transactions up to a maximum amount. The total number of transactions found is also available in the results.
