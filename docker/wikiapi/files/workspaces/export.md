# Export

Data in workspaces can be exported. To export a workspace, click the export button in the workspace overview.
You will then be presented with options to choose the type of export you want to perform.

## Workspace Export

This export type exports addresses and transactions. Additionally, it exports the structure of selectors and their settings.
This export can be used to [import](workspaces/import.md) a workspace.

File type: Json

## Entity Export

This export type exports addresses and transactions. Selectors are not exported.
The first column of the resulting file contains all transaction hashes and the second column all address hashes. 

File type: CSV
