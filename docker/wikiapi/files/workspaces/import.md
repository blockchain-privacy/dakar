# Import

A previously [exported](workspaces/export.md) workspace can be imported. In the workspace overview, click "Add Workspace" and switch to the "Import" tab to import a file.

The new workspace will appear in the workspace overview after import. Importing can take some time because all selectors must be run, the same as when creating them manually.

When the import finishes, the new workspace can be opened.

If a workspace import fails, the workspace remains usable, but some expected content may not be imported.
