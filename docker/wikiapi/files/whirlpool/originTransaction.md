# Whirlpool Origin Transaction

Whirlpool origin transactions create outputs which are used by Whirlpool mixing transactions. 

## Classification Criteria

- must have at least 3 outputs
- must have one zero amount output
- must have at least one [denominated](whirlpool/denominations.md) output (note: origin denominations also have to include funds to pay the transaction fee)
- all denominated amounts
- at least one of its outputs must be spent by a Whirlpool mixing transaction
