# Mixing Activity

The mixing activity shows visualizations of classified transactions,
which can be attributed to one address or cluster. This allows to partially observe the mixing activity of a user. Only directly connected transaction are shown.

## Bar Chart

The bar chart shows a timeline of all classified transactions. It can be filtered by transaction type and time.

## Force Graph

The force graph shows the connections between all the classified transactions. It can be filtered by transaction type and time.
