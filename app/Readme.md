# Frontend

This is the frontend of Dakar. It gives access to generic blockchain data like transactions, blocks and addresses.
Additionally, it serves as an analytics platform for CoinJoin transactions in Bitcoin and Dash.

## Dependencies

* `vue v3` - frontend framework
* `vuetify v4` - vue component library
* `d3` -  chart and graph library

For a more detailed overview check [here](./package.json).

## Front-end setup

* Install [pnpm](https://pnpm.io/installation)
* Check [package.json](./package.json) for commands.

```shell
pnpm install
```

### Compiles and hot-reloads for development
```shell
pnpm dev
```

### Compiles and minifies for production
```shell
pnpm build
```

### Starts a local server using the files produced by `pnpm build`
```shell
pnpm preview
```

### Update project

```shell
pnpm update
```
